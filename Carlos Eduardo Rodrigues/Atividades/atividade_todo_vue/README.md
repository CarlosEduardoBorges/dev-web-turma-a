# Lista de Tarefas — Vue 3 (via CDN)

Refatoração de uma lista de tarefas feita em JavaScript puro para Vue 3.

## Como usar

1. Abra a pasta `atividade_todo_vue` no seu computador
2. Clique duas vezes no arquivo `index.html`
3. O app abre direto no navegador — sem instalar nada

## Funcionalidades

- Adicionar tarefa pelo campo de texto ou pressionando Enter
- Clicar na tarefa para marcar como concluída
- Botão "x" para remover a tarefa

## Tecnologias

- HTML
- CSS
- Vue 3 (carregado via CDN, sem instalação)

## Observação

Esta é a versão simplificada, sem estrutura de projeto.
Para a versão organizada em componentes com Vue CLI, acesse a pasta:
`atividade_todo_vue (com Vue CLI)`
